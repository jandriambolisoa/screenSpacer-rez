Screen Spacer
   by Beau C. Pratt of ThePrattBros

-----------------------------------------------------------------
 
A tool for fixing the visual flicker created when a character 
is animated in stepped mode with a moving camera.

-----------------------------------------------------------------
-----------------------------------------------------------------
-----------------------------------------------------------------

Install Instructions:

1. Copy screenspacer_Script.py and screenspacer_Shelf.py to the Maya scripts folder. 

	The scripts folders are typically located in:

	(Linux)$HOME/maya/<version>/scripts
	(Mac OS X)$HOME/Library/Preferences/Autodesk/maya/<version>/scripts
	(Windows) \Users\<username>\Documents\maya\<version>\scripts

2. Copy screenspacer_Icon to the Maya icon folder

	The icon folders are typically located in:

	(Linux)$HOME/maya/<version>/prefs/icons
	(Mac OS X)$HOME/Library/Preferences/Autodesk/maya/<version>prefs/icons
	(Windows) \Users\<username>\Documents\maya\<version>\prefs\icons

3. Drag and drop the Installer file into the Maya window

The script will be installed in the Custom shelf in Maya and can be run by clicking on the icon in the Custom shelf.
